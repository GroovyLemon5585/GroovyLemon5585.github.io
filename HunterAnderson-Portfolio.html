ChatGPT Terrarium Form Webpage.


This project was for my GIMM 260 class. The goal of the project was to create a google form page, the page would then take the users answers and apply them to a premade prompt. The prompt would then be sent to ChatGPT, and then the response to the prompt would be sent back to and dislayed on the webpage. Additionally we had to create a security page in order to keep users from tampering with the answers that could be given to the prompt, and to prevent the prompt itself from being tampered with.


This project helped me to understand different coding languages, specifically HTML, CSS, and JavaScript. I was able to learn how to link two webpages, create an interactive form, setup a server, and create a page to secure information from users.

Unreal Engine Scene

This project was for my GIMM 270 class. This project involved me creating a 3D scene using Unreal Engine. all assets that were used are premade assets that I found, or created using Unreal Engine. This project was meant to help learn how to make environments using unreal engine, how to model, texture, use lighting, and how to create terrain.
